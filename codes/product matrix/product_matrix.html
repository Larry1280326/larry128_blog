Product of two matrices

Instructions

1. type the dimensions of matrix A (rows, columns).
2. type the elements of matrix one by one.
3. repeat the above proceduces for matrix B.
For example, if you want to compute \(\begin{pmatrix} 1 & 2 \\ 3 & 4 \end{pmatrix} \begin{pmatrix} 5 & 6 & 7 \\ 8 & 9 & 10 \end{pmatrix}\),
the first matrix has 2 rows and 2 columns so we type: 2 -> return -> 2 -> return,
then enter all the elements in the first matrix: 1 -> return -> 2 -> return ->3 -> return ->4 -> return.
For the second matrix, it has 2 rows and 3 columns, so we type: 2 -> return -> 3 -> return,
then enter all the elements in the second matrix: 5 -> return -> 6 -> return -> 7 -> return -> 8 -> return -> 9 -> return -> 10 -> return.
Finally, the product will be shown in the program: \(\begin{pmatrix} 21 & 24 & 27 \\ 47 & 54 & 61 \end{pmatrix}\)
